Setup Single Sign On

Using Single-Sign-On is extremely recommend since they provide better user experience and increase security.

Setting up Single Sign-On (SSO)

Entria CRM supports multiple SSO identity providers (IdP). An Identity Provider is a server that can provide identity information about users of your company. For example, Google can be an Identity Provider for companies using the G Suite solution. If your company is using G Suite you can log automatically into Entria CRM using your Google account. then a Google server will send your identity information to that site.

Azure Single Sign On

If your company uses Azure Single Sign On you'll be able to automatically login into Entria CRM using Azure as IdP. In order to use this login strategy you need to active this option in the settings menu.

Google Single Sign On

If your company uses G Suite you'll be able to automatically log into Entria CRM using Google as IdP.

SAML Single Sign on

If your company uses SAML be able to automatically log into Entria CRM SAML Servers as IdP.
