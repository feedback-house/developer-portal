Setup Single Sign On for Corporate Gmail

Using Single-Sign-On is extremely recommend since they provide better user experience and increase security.

Go inside Google Admin Apps

https://admin.google.com/ac/apps

  • Click on Saml Apps

https://admin.google.com/AdminHome?hl=en#AppsList:serviceType=SAML_APPS

  • Click on + button to create a new SAML App

  • Click on Setup my own custom app

  • Download IDP metadata, option 2 (it will be the xml used inside Entria CRM)

Application Name: Entria CRM Description: Entria CRM

You can get use one of the following icons for Entria CRM Application:

Feedback Logo with name + icon

Feedback Logo with only icon

  • Configure Service Provider

ACS URL: https://graphql.feedback.house/saml2/login/callback Entity ID https://graphql.feedback.house/saml2/metadata Name ID Basic Information Primary Email Name ID Format EMAIL

Enable SAML to everybody

Add IDP xml metadata to your Entria CRM organization

https://yourcompany.feedback.house/home/company/settings/data
